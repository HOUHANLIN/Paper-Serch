Paper-Serch 使用教程

按照步骤完成环境配置、检索、AI 总结与 BibTeX 导出，帮助你快速熟悉整个流程。

1. 基础准备

  • 安装依赖：pip install -r requirements.txt
  • 在项目根目录配置 .env（可参考 .env.example）。
  • 启动 Web 前端：python webapp.py 后访问 http://127.0.0.1:5000。

2. 配置检索

在首页的“文献检索”区域填写检索式、时间范围和最大条数。你可以直接输入 PubMed 语法，也可以在“AI 自动生成检索式”中用自然语言描述需求，点击按钮快速生成可用的检索词。

表单会根据不同数据源自动补全默认邮箱、API Key 和输出文件名，确保规范调用接口。

3. 选择 AI 模型

在“工具选择”与“AI 工具配置”区域选择模型并补充配置：

  • Gemini：填写 API Key / 模型名称（可留空使用环境变量），选择温度。
  • OpenAI：填写 API Key、Base URL（可用于自托管兼容接口）、模型名称与温度。
  • 不使用 AI 总结：选择“不使用 AI 总结”即可只生成 BibTeX。

4. 查看运行状态

提交表单后，右侧“运行进度”卡片会实时展示检索、AI 调用、BibTeX 生成等步骤的成功/失败信息，便于快速定位问题。

5. 导出与复用

  • 复制或下载 BibTeX：使用按钮复制到剪贴板或直接下载文件。
  • 查看详情：下方“文献条目”列表会显示标题、作者、期刊、摘要以及 AI 提供的中文总结与应用建议。
  • 数据不会外传：页面不会将你的检索式或密钥发送到除所选数据源和 AI 接口之外的任何地方。

6. 快捷提示

  • 若网络受限，可自定义 OpenAI Base URL 接入兼容接口。
  • 使用 AI 自动生成检索式时，可在“AI 模型”中选择实际可用的接口并填好密钥，以获得最贴合语法的结果。
  • 默认字体为 Inter / Noto Sans SC，确保在中文环境下也能获得清晰的排版效果。
